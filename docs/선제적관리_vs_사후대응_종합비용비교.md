## 3. 사후적 대응 비용

### 3.1 직접 비용 (현금 지출)

#### 3-1-1. 기본 사후관리

A·B 등급 사이트(전체의 약 0.1∼1%)에 한해 ha당 800만원/년 발생.

```
보수적 가정 (1% 면적):  800 × 5 × 0.01 = 40만원/ha
```

→ **40만원/ha**

#### 3-1-2. 자연 고사 + 복구

```
고사 본수 = 1,000본 × 1.0%/년 × 5년 = 50본/ha
복구비   = 50본 × (3만원 처리 + 10만원 식재) = 650만원/ha
```

→ **650만원/ha**

#### 3-1-3. 산불 사후 대응

```
연 평균 발생 면적 = 0.38 × 0.00010 × 5 = 0.00019 ha/ha
경제 피해 = 0.00019 × 500만원 = 0.10만원/ha
진압비 = 0.00010 × 5 × 500만원 = 0.25만원/ha
합계 ≈ 0.35만원/ha
```

→ **0.4만원/ha**

#### 3-1-4. 병해충 사후 (감염 → 고사 → 보식)

```
감염 본수 = 1,000본 × 5% = 50본/ha
고사 본수 = 50 × 30% = 15본/ha
복구비   = 15본 × (3 + 10) + 진단·방제 25만원 = 220만원/ha
```

→ **220만원/ha**

> 사용자 단순 모델 100만원/ha은 약제·인건비만 포함. 보식 비용 가산 시 220만원이 현실적.

#### 3-1-5. 산사태 사후

```
연 평균 발생 면적 = 0.25 × 0.00020 × 5 = 0.00025 ha/ha
경제 피해 = 0.00025 × 800 = 0.20만원/ha
대응비 = 0.00020 × 5 × 200 = 0.20만원/ha
합계 ≈ 0.4만원/ha
```

→ **0.4만원/ha**

### 3.2 무형 가치 손실 (사회적·환경적 비용)

#### 3-2-1. 탄소흡수량 손실 (고사목 기반)

관리 부재로 회피 가능한 고사목이 잃는 **저장탄소 + 흡수능력**. (NPP 총생산이 아닌 격리량 기준)

```
회피가능 고사 = 자연 50 + 병해충 15 = 65본/ha (위 3-1-2 · 3-1-4)
탄소 손실 = 65본 × (저장 0.35 + 5년 흡수손실 ~0.05) ≈ 25 tCO₂/ha
가격 환산 = 25 × 2만원/tCO₂ (국내 KAU) = 약 50만원/ha
```

→ **50만원/ha**

> 종전 'NPP 갭 × EU ETS'(800만원)는 NPP(총생산)를 격리량으로 혼동하고 EU 가격을 국내에 적용해 약 10배 이상 과대였음 → 격리량·국내 KAU 가격으로 정정.

#### 3-2-2. 안전사고 리스크

```
도복 사고 발생률 = 1건/100 ha (5년)
1건당 보상·소송 평균 = 1억원
기댓값 = 1/100 × 10,000만원 = 100만원/ha
+ 운영·관리 부담 (행정·민원) 가산 = 500만원/ha
```

→ **500만원/ha**

#### 3-2-3. 민원 처리 부담

```
연간 민원 건수 = 1,000본 × 0.05건/본 = 50건/ha
5년 누적 = 250건/ha
처리비용 = 250 × 3만원 + 행정 부담 = 1,500만원/ha
```

→ **1,500만원/ha**

#### 3-2-4. 자산가치 손실 (성목 손실)

```
사후 방치 시 활력 저하·고사 자산 손실
영향 본수 = 1,000본 × 5% = 50본 (관리 부재 시)
자산가치 = 50본 × 50만원 = 2,500만원/ha
```

→ **2,500만원/ha**

## 7. 정책 시사점

### 7.1 핵심 메시지

> 단순 현금 지출만 보면 사후 대응이 저렴해 보이나, **자산가치 손실(2,500만원/ha) + 민원 처리(1,500만원) + 안전사고 리스크(500만원) + 탄소흡수량 손실(50만원)** 까지 합산하면 사후적 대응의 사회적 비용은 5,460만원/ha. 선제적 관리(4,310만원/ha) 대비 **27% 비용 절감 + 무형 사회 자산 보전**.

### 7.2 DRYAD 활용

| 단계 | 기존 방식 | DRYAD 활용 |
|---|---|---|
| 위험 사이트 식별 | 민원 또는 사후 점검 | 6요인 위험점수 + 31×6 시군별 히트맵 |
| 우선순위 결정 | 담당자 경험 | A·B 등급 사이트 자동 추출 (상위 1∼5%) |
| 예산 편성 근거 | 추정 | 사이트별 추천 조치 + 예상 비용 자동 산출 |
| 정책 효과 측정 | 어려움 | 관리이력 누적 + 위험점수 변화 추적 |

특히 **A·B 등급 사이트에 집중한 선제 관리 = 사후 사회 비용 5,460만원/ha 회피**. 이를 정량적 근거로 예산 편성·심사·감사에 활용 가능.

### 7.3 활용 인용구 (사업 지원서·발표용)

> "도시숲·가로수의 사후적 대응 비용은 1 ha당 5년 누적 약 5,460만원에 달한다. 이 중 직접 현금 지출(910만원) 외에 자산가치 손실(2,500만원) · 민원 처리(1,500만원) · 안전사고 리스크(500만원) · 탄소흡수량 손실(50만원)이 사회적 비용으로 발생한다. 선제적 관리 비용(4,310만원/ha)은 이를 27% 절감하면서 무형 사회 자산을 보전한다.
>
> DRYAD는 6요인 위험점수로 A·B 등급 상위 1∼5% 사이트를 자동 식별하여, 시군 담당자가 한정된 예산을 **'어디에 집중해야 사후 사회 비용을 가장 크게 회피할 수 있는지'** 정량적으로 판단할 수 있게 한다."
